# Retail Sales Analytics Database

Retail analytics project built using PostgreSQL, Docker, and SQL that simulates large-scale retail transaction data to analyze customer behavior, revenue trends, store performance, and product sales.

## Features

- Generated large-scale retail datasets with customers, products, stores, orders, and order items.
- Analyzed top-selling products, customer spending, monthly revenue, and store performance.
- Built business-focused analytics queries for KPI reporting and sales trend analysis.
- Simulated realistic transactional retail data for portfolio-scale analysis.

## Technologies

- Database: PostgreSQL
- Tools: Docker
